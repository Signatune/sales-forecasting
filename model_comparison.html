How these scores are kept honest

Leak-free

Forecast from the past only

Every scored day is forecast from a 3-day-back origin via history_before, so no model ever sees the day it's judged on.

Same buffer

Every point forecast → P95 identically

Each candidate emits a point forecast; the evaluator applies the one uniform relative-residual buffer, so pinball measures forecast quality, not interval machinery.

Separate window

The buffer never sees its test days

Buffer residuals come from the 26 weeks strictly before the evaluation window — no scored day feeds its own P95.

Asymmetric

Pinball@95, not MAPE

An under-forecast costs 19× an equal over-forecast — the Stockout-averse asymmetry a symmetric metric can't see. MAPE is a sanity column only.
